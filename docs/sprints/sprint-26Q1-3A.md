# Sprint 26Q1-3A

## Objetivo

Consolidar la base visual de FTTH (mobile-first) para Home, búsquedas y vistas de ONT, sin integración real de datos.

## Alcance implementado

- Se creó la ruta de feature en `app/ftth`.
- Se definió layout específico de FTTH con top bar, sidebar y navegación inferior.
- Se implementó Home FTTH mobile v1.
- Se avanzó en pantallas de búsqueda FTTH, incluyendo búsqueda por elemento de red.
- Se avanzó en pantallas de detalle ONT (estructura de vista, tabs y sección de vecinos).
- Se agregaron componentes UI reutilizables para tablas, navegación y estructura visual.
- Se pulieron estilos y tokens globales en `app/globals.css` (light/dark).

## Decisiones tomadas

- Se trabajó mobile-first, alineado al diseño oficial.
- Se priorizó avance de UI y estructura antes de conectar APIs.
- Se mantuvo separación por feature (`app/ftth`) y componentes reutilizables (`components/ui`).
